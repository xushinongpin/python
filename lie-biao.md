格式

```
>>> squares = [1,2,4,5,6,7,89,0]
>>> squares
[1, 2, 4, 5, 6, 7, 89, 0]
>>>
```

索引与切片

```
>>> squares[0] #indexing returns the item
1
>>> squares[-1]
0
>>> squares[-3:] # slicing returns a new list
[7, 89, 0]
>>>
返回列表的新（浅）副本
>>> squares[:]
[1, 2, 4, 5, 6, 7, 89, 0]
>>>
串联
>>> squares + [123,345,567,789,534]
[1, 2, 4, 5, 6, 7, 89, 0, 123, 345, 567, 789, 534]
>>>

```
